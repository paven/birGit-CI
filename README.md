To set up birGit CI
set up a new git bare repositry:
Details: https://git-scm.com/book/en/v2/Git-on-the-Server-Setting-Up-the-Server

`
git init --bare
`
Install Birgit-CI in the same folder.
`
git clone git@github.com:paven/birGit-CI.git --depth 1
./birGit-CI/setup
`
